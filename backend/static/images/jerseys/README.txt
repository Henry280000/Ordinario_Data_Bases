INSTRUCCIONES PARA AGREGAR IMÁGENES DE JERSEYS
================================================

1. Coloca las imágenes de los jerseys en esta carpeta
2. Nombra los archivos según el ID del producto en MongoDB:

Nombres de archivo requeridos:
- jersey_rm_home_2024.png ✓ AGREGADA
- jersey_bar_home_2024.jpg (o .png)
- jersey_man_home_2024.jpg
- jersey_liv_home_2024.jpg
- jersey_psg_home_2024.jpg
- jersey_bay_home_2024.jpg
- jersey_ame_home_2024.jpg (Club América) - NUEVO
- jersey_dor_home_2024.jpg (Borussia Dortmund) - NUEVO

Formato recomendado:
- JPG o PNG
- Tamaño: 800x800px (cuadrado)
- Fondo transparente o blanco
- Peso: máximo 500KB por imagen

Si no tienes las imágenes, el sistema mostrará un placeholder.
